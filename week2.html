Week 2

Ok, I'm going to be honest. This is the week that I was the most scared about. I feel fairly comfortable with the programming and database stuff, but statistics less so. I've taken Linear Algebra, Multivariable Calculus, and Statistics in college, but haven't put it to use in years. So here we go, week 2.

Day 1 - Probability Review

The morning started off with some probability questions. Conditional probability (Bayes' Theorem), the chain rule, combinations, permutations, etc.

Next, distributions. Bernoulli, Binomial, Poisson, Geometric, Gaussian, Uniform, Exponential, etc.

The pair exercise involved determining the best distributions to model certain scenarios. This is something I haven't really done, so it took me a bit to get up to speed.

We learned about scipy.stats, and how horrendous the documentation is. It's pretty great to be able to look at some data, and then 'create' distributions with certain parameters. Answering the question "we've got data, what distribution does it follow?" allows us to do a lot.

In scipy.stats we modeled distributions, their pdfs/pmfs/cdfs, created rvs's, computed correlation/covariance, and looked at percentiles.
